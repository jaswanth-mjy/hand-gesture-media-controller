## How to Contribute

### Reporting Bugs

If you find a bug, please open an issue with:
- Clear description of the problem
- Steps to reproduce
- Expected vs actual behavior
- Your environment (OS, Python version, webcam model)
- Screenshots or error messages if applicable

### Suggesting Features

Feature requests are welcome! Please:
- Check existing issues first
- Describe the feature clearly
- Explain the use case
- Include examples if possible

### Pull Requests

1. **Fork the repository**
   ```bash
   git clone https://github.com/jaswanth-mjy/hand-gesture-media-controller.git
   cd hand-gesture-media-controller
   ```

2. **Create a branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **Make your changes**
   - Follow existing code style
   - Add comments for complex logic
   - Test thoroughly

4. **Commit your changes**
   ```bash
   git add .
   git commit -m "Add: description of your changes"
   ```

5. **Push and create PR**
   ```bash
   git push origin feature/your-feature-name
   ```
   Then open a Pull Request on GitHub

## Code Style

- Follow PEP 8 guidelines
- Use meaningful variable names
- Add docstrings to functions
- Keep functions focused and small
- Comment complex algorithms

## Testing

Before submitting:
- Test with different lighting conditions
- Try various hand positions
- Check with multiple media players
- Verify on different OS if possible

## Ideas for Contribution

- Add new gesture types (volume control, skip)
- Improve detection accuracy
- Add GUI configuration panel
- Support for multiple hands
- Better error handling
- Performance optimizations
- Documentation improvements
- Video tutorials

## Questions?

Feel free to open an issue for any questions!
